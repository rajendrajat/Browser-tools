Tests use fetch() with mode: 'no-cors' + HEAD requests from your browser. Reachable = network path is open. Blocked = connection-level failure (firewall / proxy / DNS). Redirected = server responded but forwarded elsewhere. Opaque responses (status 0) from no-cors fetches are counted as reachable — this is expected for non-CORS APIs.
